This extension includes 2 flexible treemenus with various display options 

Edit treemenu.ini.append to achieve different menu effects

Treemenu 1
-------------------

Authors: Björn Dieding

You can see a working sample of treemenu1 on http://www.xrow.de

Install the extension and include it in your pagelayout.tpl like this:

{include uri="extension/treemenu/design/standard/templates/treemenu1.tpl"}
